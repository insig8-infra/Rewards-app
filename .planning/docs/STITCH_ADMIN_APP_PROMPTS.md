# Stitch Admin App Prompts

**Status:** Canonical prompt pack lives in client deliverables.

Use:

- `client-deliverables/Stitch_Admin_App_Prompts.md`

This planning pointer exists so future planning/build sessions can discover the admin mobile Stitch prompt pack without duplicating content.

The admin prompt pack is based on the 2026-06-13 Volt Admin clarification:

- Admin app name: `Volt Admin`.
- Personas: OWNER and STAFF.
- Bottom nav: Dashboard, Return Scan, Contractors, More.
- OWNER can register/manage contractors and staff.
- STAFF can view contractors/reports, Cancel unused/uncollected returned-product QR codes, Reverse already collected/scanned returned-product QR codes, and fulfill verified reward claims.
- Reward fulfillment requires Claim ID plus OTP sent to the registered contractor mobile number.
- Contractors bottom nav opens Contractor Leaderboard first, with manage-contractor routes below it.
- QR printing remains a web portal workflow.
