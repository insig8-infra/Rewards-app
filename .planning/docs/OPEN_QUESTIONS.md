## Confirmed Answers Logged

- QR tracking is at individual QR/unit level, not only line-item level.
- Staff can later print remaining `Not Printed` units in multiple partial batches.
- If returned QR points were claimed but not fulfilled, system should unclaim/revoke the claim.
- If returned QR points were already fulfilled, reversal can create a negative contractor balance.
- Cancel/Reverse reason is fixed as `Product Returned`; no extra proof is required.
- QR expiry is 45 days for now.
- QR reprint uses a replacement QR token and invalidates the old QR token.
- Helper Recent contractor is stored locally on the helper's device using secure device storage.
- Helper Recent contractor is convenience-only and OTP remains required for every helper login/session.
- Admin app working name is `Volt Admin`.
- Admin mobile has OWNER and STAFF personas, superseding the earlier one-admin-role assumption.
- OWNER logs in with mobile number and fixed 4-digit PIN.
- STAFF logs in with mobile number and assigned 4-digit PIN generated by the app.
- OWNER and STAFF sessions persist unless access is removed/deactivated, PIN changes, or the app is not opened for 4 straight days.
- Admin app supports English/Hindi through a Profile language setting, not per-screen toggle.
- Admin mobile bottom nav is Dashboard, Return Scan, Contractors, More.
- OWNER can register contractors from admin mobile with name, mobile number, and optional photo.
- ContractorID is auto-generated during admin mobile registration.
- Contractor registration sends welcome SMS with frontend app download links.
- Duplicate contractor mobile number is blocked and shows existing contractor details.
- OWNER can edit and deactivate/unregister contractors.
- STAFF can view contractor list/details but cannot add, edit, deactivate, unregister, delete profiles, or manually change points.
- OWNER can add staff members, reset/regenerate staff PIN, deactivate staff, and reactivate staff.
- Cancel applies only to unused/uncollected, Printed/Unclaimed, non-expired QR codes and does not show contractor or points details.
- Reverse applies only to already collected/scanned, Scanned/Claimed QR codes and reverses points before discarding the QR.
- Cancel and Reverse both require confirmation that the QR label was removed and discarded.
- Reverse confirmation warns if reversal may create a negative contractor balance.
- OWNER and STAFF can fulfill reward claims by Claim ID plus OTP sent to the registered contractor mobile number, then mark the claim `Fulfilled`.
- Admin mobile reports include QR printed by date/category, contractor leaderboard, QR status, reward claims, returns/reversals, and product/category performance.
- Admin mobile Contractors bottom nav opens Contractor Leaderboard first, while Reports Hub exposes the report/export route for the same leaderboard data.
- STAFF reports are view-only.
- OWNER can export/share reports as PDF, Excel, and WhatsApp.
- Admin mobile notifications/alerts are not needed for v1.
